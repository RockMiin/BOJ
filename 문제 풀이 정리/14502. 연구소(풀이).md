**search_wall()**

```python
def search_wall(cnt):

    if cnt==3:
        bfs()
        return

    for i in range(n):
        for j in range(m):
            if adj[i][j]== 0:
                adj[i][j]= 1
                search_wall(cnt+1)
                adj[i][j]= 0
```

벽을 3개 세웠을 시에 bfs 호출

아래 반복문은 벽을 3개 세우는 과정임. 벽을 하나 세우고 값을 변경해주고, 함수를 재귀적으로 호출하는 방식으로 구현. 더 이상 벽을 세울 곳이 없으면 함수를 빠져나와서 adj 값을 다시 변경해 준 다음에 다음 위치를 탐색한다.
